# Kapcsolatfelépítés

---

# HTTP

- Egyszerű szüveges protokoll
- Kliens-server architecture
  - Kliens kezdeményez, szerver válaszol
- Független kérések
 
---
 
# HTTP kérés

- Kérés (kérés célja, URL)
- Fejléc
  - Preferált nyelv, kliens információk
  - Host mező: a kért oldal hosztneve
- Opcionális adat (űrlap, feltöltött fájl)
 
---
 
# HTTP kérés fajták
 
- GET
- POST
- PUT
- DELETE
- ...

---

# GET
- Egyszerű kérés, adatküldés nélkül
- URL-ben küldhetünk paramétereket

```/search?query=kir-dev```

---

# POST
- Adatküldés (bejelentkezés, feltöltés)

---

# HTTP válaszok

- Három rész
  - Státuszkód
  - Fejléc (legutóbbi módosítás, szerver információk)
  - Opcionális adat (HTML oldal, letöltött fájl)

# CSS selectorok

```
div, p { }                //div és p
div p { }                 //div-ben p
div > p { }               //p közvetlen szülője egy div
div + p { }               //div utáni p (közvetlen testvér)
div ~ p { }               //div utáni p (testvér)
input[type=”text”] { }    //input type attributuma text
```
